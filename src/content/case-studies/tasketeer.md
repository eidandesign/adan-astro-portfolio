overview: The client wanted a project where it would be easy to find internal company documents. The main objective was to use AI to access the documentation, but with the security and privacy that companies need. A character was implemented to serve as the user’s assistant, similar to the old Clippy from Word, but this time using AI and adjusting it with LLM to perform each task more efficiently.
